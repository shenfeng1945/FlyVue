### InfiniteScroll 无限滚动

### InfiniteScroll Attributes

| 参数 | 说明 | 类型 | 可选择 | 默认值 |
| -- | -- | -- | -- | -- |
| infinite-scroll-delay | 节流延时，单位为ms | number | - | 200 |
| infinite-scroll-distance | 触发加载的距离阈值，单位为px | number | - | 0 |
